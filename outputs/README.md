# Outputs

This folder contains output artifacts generated from the notebooks.

## Recommended Structure

- `outputs/figures/`  
  Store plots such as:
  - topic trend plots
  - forecast plots
  - volatility plots
  - model comparison plots

- `outputs/results/`  
  Store result files such as:
  - forecasted topic values
  - evaluation metrics (MAE, RMSE, MAPE, DA)
  - topic keyword summaries
