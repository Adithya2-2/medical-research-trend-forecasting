# Notebooks

This folder contains the implementation notebooks for the project.

## Files

### `nlp_code-1.ipynb`
Implements the NLP + topic modeling pipeline:
- Dataset loading
- Text preprocessing
- SBERT embeddings generation
- Topic modeling using BERTopic

### `nlp_2.ipynb`
Implements the forecasting + evaluation pipeline:
- Monthly topic time-series formation
- Forecasting using ARIMA, Prophet, and LSTM
- Evaluation using MAE, RMSE, MAPE, and Directional Accuracy
- Plot generation (trend, forecast, volatility)
